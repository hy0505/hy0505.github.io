# 2019-07-03 营销兑换券组件

# 功能

  • 提供添加营销兑换券弹框
  • 目前只支持填写名称、有效期和选择商品
  • 兑换券名称会根据target_name生成为[target_name]-名称

# 属性参数

属性名 默认值 备注
isShow false 是否显示弹框
exchangeCertificateId "" 兑换券id。传入此值为编辑状态
target_from "" 来源。例如：lottery.xxx
target_name "" 活动名称。如抽奖活动，就传抽奖。用于生成兑换券名字
autoAdd true 保存是否自动提交api。true：自动调接口，并返回对应的数据。false：直接emit所有的数据
